##Project set up instructions
Follow these steps to set up and run the project locally:

1 **Clone the repository**
   git clone https://github.com/kamarushem-necitizen/My-Nature-Photography.git
2 cd "My Nature Photography"  to navigate into the project folder. 
3 Open the project in VS Code.
4 Run the site locally by opening index.html. And it will open with styling from style.css.
